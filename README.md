# Cards Game Telegram bot

## Installation

Execute:

```shell
$ bundle
```

## Configuration

First things first, you need to [obtain a token](https://core.telegram.org/bots#botfather) for your bot. 
Edit `settings.rb` with:

```ruby
token = 'YOUR_TELEGRAM_BOT_API_TOKEN'
```

## Run

```ruby
ruby ilegal_words_bot.rb
```
